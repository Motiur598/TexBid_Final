# TexBid — RFQ System

## What is an RFQ?
A Request for Quotation (RFQ) is posted by a buyer to invite suppliers to submit bids for a garment order.

## Creating an RFQ (Buyer)
1. Hover over **RFQ** in the nav → click **Create RFQ**
2. Fill in: title, product category, quantity, fabric type, urgency, deadline
3. Optionally enable **Reverse Auction** and set auction end time
4. Submit — RFQ appears in Open RFQs feed

## Browsing RFQs (Supplier)
- Nav → **RFQ** → **Browse RFQs**
- Filter by category or urgency
- Click **View Details** to see full RFQ
- Click **Place Bid** or **Submit a Bid** to bid

## RFQ Detail Page
- Shows: basic info, quantity breakdown, specifications, timeline, certifications
- **Posted by**: buyer company name shown for clarity
- **Reverse Auction Details**: shows auction end time + Join Auction Room (Premium only)
- **Bids section**: visible to all logged-in users; rejected bids hidden from non-owners

## Bid Rules
- Only the **RFQ owner (buyer)** can accept bids
- Only the **latest bid per supplier** is shown (previous bids hidden)
- Rejected bids are **only visible to the buyer** — suppliers never see them
- Free users: 1 bid per RFQ. Premium users: unlimited bids

## RFQ Statuses
| Status | Meaning |
|--------|---------|
| OPEN | Accepting bids |
| EVALUATING | Buyer accepted a bid, awaiting supplier confirmation |
| AWARDED | Payment completed |
| CLOSED | Manually closed |

## Editing / Deleting an RFQ
- Only the RFQ owner can edit or delete
- Edit button and Delete button appear in the top-right of the RFQ detail page for owners
